# Libjasper Package

Package: `libjasper/libjasper`

License: JasPer-2.0

Install with `pnl`:

```sh
pnl install https://github.com/m3m0r7/pnl-packages/tree/main/packages/libjasper
```

## PHP usage

```php
use Pnlx\Libjasper\Libjasper;

// jas_getversion() returns the JasPer JPEG-2000 library version string.
echo Libjasper::jas_getversion() . PHP_EOL;
```
