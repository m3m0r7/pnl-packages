# libopenjp2/libopenjp2 examples

```php
use Pnlx\Libopenjp2\Libopenjp2;

// opj_version() returns the OpenJPEG version string, e.g. "2.5.2".
$version = Libopenjp2::opj_version();
echo "OpenJPEG: $version\n";
```
