# GMP Package

Package: `libgmp/libgmp`

Base library: GMP

License: LGPL-3.0-or-later

Install with `pnl`:

```sh
pnl install https://github.com/m3m0r7/pnl-packages/tree/main/packages/libgmp
```

The package requires GMP headers and library files to be available through the system, Homebrew, pkg-config, or configured search paths.
