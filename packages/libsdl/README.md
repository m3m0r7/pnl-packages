# SDL Package

Package: `libsdl/libsdl`

Base library: SDL2

License: Zlib

Install with `pnl`:

```sh
pnl install https://github.com/m3m0r7/pnl-packages/tree/main/packages/libsdl
```

The package requires SDL2 headers and library files. The examples in `test2.php` and `test3.php` open a small SDL window titled `Hello World!`.
